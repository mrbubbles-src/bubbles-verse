## Schnellstart

- Die Schnellstart-Aktionen auf `/` sind aktuell auf den Coding Vault
  gemünzt.
- Später braucht jede Aktion eine App-Auswahl, sobald Portfolio und weitere
  Apps eigene Erstellungs- oder Pflegeflows im Dashboard haben.
- Erwartetes Verhalten:
  - `Neuer Eintrag` öffnet zuerst eine App-/Content-Type-Auswahl, wenn mehr
    als ein Ziel möglich ist.
  - `Einträge öffnen` kann zwischen App-Bereichen filtern.
  - `Kategorien öffnen` wird nur für Apps angezeigt, die Taxonomie pflegen.
  - `Zugangsverwaltung` bleibt global.
